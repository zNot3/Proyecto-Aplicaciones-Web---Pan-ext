# Pan-ext

Una aplicación Android y Web educativa para la salud y trackeo de consumo, que ayuda a poder vivir una vida mas saludable y estar por encima de tus gastos alimenticios.

## Funciones

- Gestión de Inventario
- Generador de Recetas con IA (Gemini Integration):
  - Plan: Enviar a Gemini la lista actual de ingredientes del usuario. 
  - Prompt: "Actúa como un Chef experto. Basado en estos ingredientes: *lista*, genera 3 opciones de recetas saludables". 
  - o Resultado: La App mostrará las recetas sugeridas sin que el usuario tenga que buscar en internet. 
- Lista de Compras Inteligente
- Alertas de Caducidad
- Recetas "Paso a Paso"
